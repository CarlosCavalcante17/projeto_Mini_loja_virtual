# projeto_Mini_loja_virtual

Este é um projeto de uma mini loja virtual desenvolvido com PHP, HTML, CSS e JavaScript, utilizando o XAMPP como ambiente de desenvolvimento local. O sistema básico inclui funcionalidades de cadastro de usuários, autenticação (login/logout), exibição de produtos, carrinho de compras e acompanhamento de pedidos.

✨ Funcionalidades
Autenticação de Usuário: Login e Logout de clientes.

Gestão de Sessão: Manutenção do estado do usuário logado (usando $_SESSION).

Visualização de Produtos: Página inicial com listagem de itens.

Carrinho de Compras: Adicionar, remover produtos no carrinho

Página de Pedidos: Visualização do histórico de compras do usuário logado (meus_pedidos.php).

Conexão com Banco de Dados: Utilização do MySQL para persistência de dados.

⚙️ Tecnologias Utilizadas
Backend: PHP (Linguagem principal)

Banco de Dados: MySQL (Gerenciado via phpMyAdmin no XAMPP)

Frontend: HTML5, CSS3, JavaScript

Ambiente Local: XAMPP
